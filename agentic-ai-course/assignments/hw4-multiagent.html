HW4 — Multi-Agent Systems

Released: April 3  ·  Due: April 20, 11:59pm  ·  Points: 100

Overview

Build a multi-agent pipeline using LangGraph in which multiple LLM-powered agents collaborate (and potentially compete) to solve a complex task. Corresponds to AIMA 4th ed. Chapter 18 and supplemental readings on LLM orchestration.

Tasks

  1. Cooperative Research Pipeline (40 pts) — Implement a 3-agent system: a Planner, a Researcher, and a Writer. The system must produce a structured 500-word report on an assigned topic using web search tools.
  2. Debate System (30 pts) — Implement a 2-agent debate: one agent argues for, one against a proposition. A judge agent (separate LLM call) scores each round. Run 3 debate rounds on a provided topic.
  3. Analysis (30 pts) — For each system, measure quality (human-rated rubric provided) and cost (total tokens). Discuss trade-offs between number of agents, quality, and cost.

Submission

Submit via Moodle as hw4-multiagent.zip. Include a results/ folder with your generated reports and debate transcripts.
