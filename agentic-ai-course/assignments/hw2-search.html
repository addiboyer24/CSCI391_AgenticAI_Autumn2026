HW2 — Search & Planning

Released: February 4  ·  Due: February 23, 11:59pm  ·  Points: 100

Overview

This assignment covers classical search and planning. You will implement several search algorithms and a basic STRIPS planner, applying them to toy planning domains. Corresponds to AIMA 4th ed. Chapters 3, 4, and 11.

Tasks

  1. BFS, DFS, and A* (30 pts) — Implement in search/algorithms.py and test on the 8-puzzle.
  2. Heuristic Design (20 pts) — Implement and compare two admissible heuristics for A*. Measure nodes expanded.
  3. Adversarial Search (20 pts) — Implement Minimax with alpha-beta pruning for a 4×4 Tic-Tac-Toe variant.
  4. STRIPS Planner (30 pts) — Implement forward-chaining planning for the Blocks World domain. Evaluate on 5 provided problem instances.

Submission

Submit via Moodle as hw2-search.zip.
