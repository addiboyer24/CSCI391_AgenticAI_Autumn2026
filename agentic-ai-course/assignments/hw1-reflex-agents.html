HW1 — Reflex Agents

Released: Week 1  ·  Due: February 9, 11:59pm  ·  Points: 100

Overview

In this assignment you will implement several simple reflex agents operating in a grid-world environment, then analyze their performance against environment properties (fully/partially observable, deterministic/stochastic). The goal is to build intuition for the PEAS framework and the distinction between agent types described in AIMA Chapter 2.

Learning Objectives

  • Implement simple-reflex and model-based reflex agents in Python.
  • Characterize an environment using the PEAS framework.
  • Measure agent performance using a custom performance measure.
  • Understand the limitations of reflex agents in partially observable environments.

Getting Started

Download the starter code from Moodle (hw1-reflex-agents.zip). Extract and install dependencies:

cd hw1-reflex-agents
pip install -e .
python run_agent.py --agent simple_reflex --env vacuum

Tasks

  1. Implement SimpleReflexAgent (30 pts) — Complete agents/simple_reflex.py. The agent receives a percept and must return an action using only condition-action rules.
  2. Implement ModelBasedReflexAgent (30 pts) — Complete agents/model_based.py. Maintain an internal state model to handle partial observability.
  3. PEAS Analysis (20 pts) — In report/peas_analysis.md, describe the PEAS for two environments of your choice from the provided set.
  4. Performance Comparison (20 pts) — Run both agents across 100 random episodes in the stochastic vacuum world and report mean performance. Write up your findings in report/analysis.md.

Submission

Submit via Moodle: a single .zip of your project directory. Ensure your code runs from a fresh environment install. Include your two report markdown files.

Reading

Russell & Norvig, AIMA 4th ed., Chapter 2 (all sections).
