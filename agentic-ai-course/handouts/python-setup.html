Python & Environment Setup

All programming assignments use Python 3.11+. We recommend managing dependencies with conda (via Miniforge or Anaconda).

1 — Install Miniforge

Download and install Miniforge for your OS (Windows, macOS, Linux). Then verify:

conda --version

2 — Create the Course Environment

conda create -n agentic-ai python=3.11 -y
conda activate agentic-ai

3 — Install Core Packages

pip install numpy scipy matplotlib pandas jupyter \
    openai anthropic langchain langgraph \
    gymnasium stable-baselines3 torch torchvision

4 — API Keys

Several assignments use LLM APIs. Store your keys as environment variables — never hard-code them or commit them to git.

# macOS / Linux — add to ~/.zshrc or ~/.bashrc
export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."
# Windows PowerShell
$env:OPENAI_API_KEY="sk-..."

5 — Verify

python - <<'EOF'
import openai, anthropic, gymnasium, torch
print("All packages imported successfully.")
print("PyTorch version:", torch.__version__)
EOF

Troubleshooting

  • On Apple Silicon, install PyTorch with pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu for CPU-only, or follow the official MPS guide.
  • If conda is slow, switch to mamba (included with Miniforge): replace conda with mamba in any command.
  • Post in #setup-help on Discord with your OS, Python version, and the full error traceback.
